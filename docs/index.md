# Kyntex Tendon Stiffness Monitor

Kyntex is a wearable embedded system for estimating relative patellar tendon stiffness using controlled mechanical excitation and synchronized accelerometer sensing.

## Demo

Coming soon.

## Hardware

- nRF54L15
- Piezo actuator
- Synchronized accelerometers
- ADS131M02 / ADS131M08 ADC

## Project Goals

- Estimate relative tendon stiffness
- Compare left vs right tendon trends
- Track pre/post exercise stiffness changes
- Stream data over BLE
